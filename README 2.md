### Local development

To run the interactive presentation locally:

1. Install the dependencies

```bash
$ yarn
```

2. Run the app

```bash
$ yarn present
```

## Demos

The `/demos` folder contains all the example code, along with working Jest tests. To run those:

1. Install the dependencies

```bash
$ yarn
```

2. Run the tests

```bash
$ yarn test
```
